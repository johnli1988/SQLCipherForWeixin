# SQLCipherForWeixin
原版的软件有bug,即使有微信的数据库密码(16进制)也无法打开数据库，所以从[github](https://github.com/sqlitebrowser/sqlitebrowser.git) clone 了代码,做了一些修改，可以支持16进制的微信数据库密码。并且只有一个可执行文件。
